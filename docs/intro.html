SheetMusic

SheetMusic is a simple program written in Python and QT to display music on a monitor. It provides bookmarks, single and double page display, simple navigation, and tools to convert PDF documents into its internal format.

Documentation is available for most of the features. SheetMusic is open source, and is being updated. If you have a feature you'd like to add, feel free to clone the source and make changes.
