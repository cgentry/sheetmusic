SheetMusic is a simple program written in Python 3 and PySide6 (Qt) to display music on a monitor. It provides bookmarks, single and double page display, simple navigation, and tools to convert PDF documents into its internal format.

Help is being added as it is being developed. SheetMusic is free, open source, and can be used as-is or modified to provide features you prefer. You can make suggestions, but there is no guarentee that the feature will be added, or when it might be added.
