Page Numbering

PDFs often have a title page and, sometimes, several pages of introduction or copyright pages. After these initial pages you will find the pages will display page numbers. In order to reduce confusion, the program introduces a concept of an 'offset to page one', i.e. the page where numbering starts.

Page display

A page number is always displayed in the lower right of the status bar. This number can be one of two values:

  • Book page This is the absolute page number. If an offset is set, this is a page that occurs before the page numbering starts in the book. For example, if there are 4 pages before page numbering starts, and you've set the offset to 4, any page in the book before page 4 will have the label of 'Book page'.

Setting the page numbering offset

There are two ways to set the page numbering offset: when you import the document, or in the document properties option. Either method does the same thing.

File Import When you import the PDF you are prompted to enter the offset to the first page.


Book's Properties

The book's properties can be altered at anytime once a book is open. If the book's page 1 is being displayed on the screen but the pae number in the lower left doesn't match, or it is displaying 'Book page', set the page offset to match the Book page. First select 'Edit' from the menu bar, then 'Properties'. A dialog box will open and you can alter the 'Offset to page 1'.
